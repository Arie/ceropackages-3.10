Getting Help

This is the "Ocean City" (July 15, 2011) release of CeroWrt. Please see the main Cerowrt Ocean City Wiki, Installation Guide, Release Notes, and FAQ for more detailed and up to date information.

Known bugs about Uberwrt and Cerowrt are kept in the UberWrt bug database.

Support for CeroWrt is available primarily on the #bufferbloat irc channel on chat.freenode.net, and there may one day be a #cerowrt specific channel. If you do not have an irc capable client already, you can get on irc chat via this convenient web page.

Bufferbloat related problems are discussed on the bloat and bloat-devel mailing lists. There are comprehensive archives of those lists available as well.

For more information on UberWrt in general, please see our project wiki.
