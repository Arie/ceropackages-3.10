Features in this release

These features are all installed by default:

  • Extensive Debloating
  • Numerous debugging and diagnostic tools
  • Multiple TCP algorithms (Cubic,Bic,Westwood+,Vegas)
  • Multiple traffic shapers (now including DRR and SFB)
  • Simulations possible of packet loss and delay with NETEM
  • Native, 6to4, and 6in4 IPv6 support
  • Mesh routing
  • The polipo web proxy
  • Local Web Server & Rsync

Many additional packages not installed by default, but are available in the cerowrt package repository.
