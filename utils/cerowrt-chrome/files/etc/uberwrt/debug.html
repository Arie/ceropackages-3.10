Debugging and diagnostic tools

Most home routers shipped to date have very primitive, or absent network testing and debugging tools. CeroWrt comes with a rich set of such, some critical ones are installed by default, and many more available in the package repository.

Default Packages

    tcpdump,traceroute,ping,ping6,FIXME

Optional Packages

These include, but are not limited to:

    iperf,lft,netperf,httpping,bing,FIXME

Update and review the package listing in the router's administrative interface for a complete listing.

The cerowrt-dbg releases also contain oprofile support, which is very useful for trying to see problems integral to the kernel itself and networking stack. Due to the "heisenbugging" performance impact of oprofile, however, it is not available in the primary releases of CeroWrt.
