Interesting features of this release

  • Extensive debloating using CoDel, fq_codel, and sfq_codel
  • DNS services using dnsmasq
  • mDNS services to provide DNS names for local devices so you don't have to know their IP address
  • Numerous debugging and diagnostic tools
  • ECN is enabled
  • Native, 6to4, and 6in4 IPv6 support
  • Mesh routing
  • LuCI web interface
  • The polipo web proxy
  • Local lighttpd web server
  • Rsync
  • CeroWrt routes between its interfaces (which have different /27 subnets by default), rather than bridging traffic between them. Bridging different radios and Ethernet has become very problematic, particularly in the face of multicast traffic and radically different wired/wireless bandwidth.

Many additional packages are not installed by default, but are available in the CeroWrt package repository.
