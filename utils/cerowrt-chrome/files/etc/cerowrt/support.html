Getting Help

This is the "Ocean City" (Sept 15, 2011) release of CeroWrt. Please see the main CeroWrt Ocean City Wiki, Installation Guide, Release Notes, and FAQ for more detailed and up to date information.

Known bugs about CeroWrt are kept in the CeroWrt bug database.

Support for CeroWrt is available primarily on the #bufferbloat IRC channel on chat.freenode.net, and there may one day be a #cerowrt specific channel. If you do not have an IRC capable client already, you can get on the bufferbloat IRC channel via this convenient web page.

Bufferbloat related problems are discussed on the bloat and bloat-devel mailing lists. There are comprehensive archives of those lists available as well.

For more information on CeroWrt, please see our project wiki.
